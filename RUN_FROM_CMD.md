### HOW TO RUN FROM CMD
1] Navigate to the folder which has all project files.
2] Type "cmd" into the file directory/search bar.
3] Once cmd opens, type "g++ -std=c++17 -O2 main.cpp -o nro" and hit enter.
4] After it has finished compiling, type "nro" and hit enter.
### IF YOU WANT TO CHANGE THINGS
- If you want to change the A* Heurstic, GWO Parameters, or Graph size, open up "main.cpp", 
and you can find the initial conditions easily.
